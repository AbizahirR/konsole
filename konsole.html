Konsole

A small library to format the console output in the browser using css

How to use it:

In your javascript script:

  • konsole.log: the original console.log

    konsole.log("this is an original console.log command using Konsole library")

  • konsole.notice: a way to show some informartion that need to be emphasized but is not that important

    black font on light green background
    konsole.notice("A NOTICE is for small pieces of information not that relevant")

  • konsole.info: if you want to show important information on the console

    white font on blue background
    konsole.info("Information:\n", "just keep and eye on this message, it could be important")

  • konsole.warning: Alert, something is happening

    black font on yellow background
    konsole.warning("WARNING: a potential problem or a very important information is here")

  • konsole.error: Houston, we have a problem!

    white bold font on red background
    konsole.warning("WARNING: a potential problem or a very important information is here")
